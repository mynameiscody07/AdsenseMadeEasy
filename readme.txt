== Description ==

Adsense Made Easy is a very simple plugin that allows the user to add Adsense to any widgetized area of the current theme.

You have full control over google ads.  Every option available from the google adsense website is available in this widget, making the process incredibly easy and quick.

To make money with Google Adsense ads you just have an account with Google Adsense.

**[Download](http://downloads.wordpress.org/plugin/adsense-made-easy.0.5.1.zip)**

Features:

* Easily create and style Google Adsense Ads
* **Earn money** with Google Adsense ads.
* This plugin will not work properly if used on site with adult/pornographic content. 
* The plugin is campatible with the widget system.
* Easy to use and understand widget panel.
* Limited to no programming knowledge required to use this plugin
* Through a simple menu you can choose the size, colors, and type of ads used.
* Capable of adding custom channels for easy tracking in the Google Adsense Interface.

== Installation ==

1. Download the current version of **Adsense Made Easy** from the download link on the right.

2. Upload the folder `adsense-made-easy.zip` to the directory `/wp-content/plugins/` in your wordpress setup.

3. Activate the plugin Adsense Made Easy through the 'Plugins' menu in the administration page of your WordPress.

4. Head to the Widget section of your Appearance Options and use the Adsense Made Easy Plugin.

== Frequently Asked Questions ==

= How do I **Configure/Use** this plug in? =

Adsense Made Easy is meant to be a very simplistic Adsense plugin.  To use Adsense Made Easy follow the Installation Instructions.
Click on your widgets admin panel under the appearance tab.  Drag and drop the Adsense Made Easy widget into your desired widget area.
Next add your Publisher ID to the widget and configure the options available.

= Will Adsense Made Easy be supported and upgraded? =

Yes, Adsense Made Easy was created to make adding Adsense to any wordpress widgetized area very simple and easy.
Features will be continuously added to make Adsense Made Easy the best Adsense Widget Plugin around.

= Can I use  this plugin at adult/pornography sites? =

No, you can't use this plugin to show ads at adult/pornography sites. As stated in Google Adsense program policies. AdSense publishers are not permitted to place Google ads on pages with pornographic content.
This includes photos or videos which contain full nudity or sexual activities. More information about this policy can be found in Adsense help center (https://www.google.com/adsense/support/as/bin/answer.py?hl=en&answer=105957).
